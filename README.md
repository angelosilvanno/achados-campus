# 🎓 AchadosCampus

O **AchadosCampus** é um sistema web moderno e responsivo desenvolvido para facilitar a recuperação de objetos perdidos dentro de instituições de ensino. O diferencial do sistema é o conceito de **Match Inteligente**, que cruza dados de quem perdeu com quem encontrou em tempo real.

## 🚀 Tecnologias Utilizadas
- **React 18** com **Vite**
- **TypeScript** (Tipagem estrita)
- **Tailwind CSS v4** (Design moderno e utilitário)
- **React Router Dom** (Navegação dinâmica)
- **Context API** (Gerenciamento de estado global do usuário)

## 📋 Funcionalidades Principais
- [x] Login e Registro Institucional
- [x] Registro de Objetos Perdidos com descrição detalhada
- [x] Registro de Objetos Achados com upload de fotos
- [x] Mural de Itens centralizado para consulta
- [x] Fluxo de navegação responsivo (Mobile/Desktop)

## 🛠️ Como rodar o projeto
1. Clone o repositório: `git clone https://github.com/angelosilvanno/achados-campus.git`
2. Instale as dependências: `npm install`
3. Inicie o servidor de desenvolvimento: `npm run dev`
